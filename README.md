# 思源笔记模板集

本项目是思源笔记的模板集合，用于提升笔记整理和文档聚合的效率。

## 安装

在思源笔记 `[根目录]/data/` 目录下执行以下命令：

```bash
git clone <本项目地址>
```

然后将项目文件夹重命名为 `templates` 即可。

## 模板文件介绍

### 第一类：1.x - 子文档框架

用于创建子文档时提供标准化的内容结构，便于父文档集成时快速了解子文档的主要内容。

| 文件 | 说明 |
|------|------|
| [1. 子文档框架.md](1.%20子文档框架.md) | 基础子文档模板，包含摘要和正文结构 |
| [1.1 会议纪要.md](1.1%20会议纪要.md) | 会议纪要专用模板，包含时间、参会人、会议结论等字段 |
| [1.2 供应商技术交流记录.md](1.2%20供应商技术交流记录.md) | 技术交流记录模板，包含公司介绍、功能框图、业务模式、成功案例等结构 |

### 第二类：2.x - 父文档目录

用于父文档快速获取子文档列表及其摘要，支持增量更新（已存在的目录不会重复添加）。

| 文件 | 说明 |
|------|------|
| [2.1 父文档目录（倒序）.md](2.1%20父文档目录（倒序）.md) | 按创建时间倒序排列子文档 |
| [2.2 父文档目录（正序）.md](2.2%20父文档目录（正序）.md) | 按创建时间正序排列子文档 |

### 第三类：3.x - 按标签读取内容块

基于标签聚合相关内容，自动将含指定标签的文档或内容块汇总到父文档中。

| 文件 | 说明 |
|------|------|
| [3. 按内容标签聚合（正序）.md](3.%20按内容标签聚合（正序）.md) | 按文档的第一个标签聚合相关文档，支持正序排列 |

### 第四类：8.x - Markdown 代码片段

思源笔记常用的 Markdown 代码小模板。

| 文件 | 说明 |
|------|------|
| [8.1 插入视频.md](8.1%20插入视频.md) | 插入 HTML5 视频播放器的代码片段 |
| [8.2 ID引用（不含块Title）.md](8.2%20ID引用（不含块Title）.md) | 通过块 ID 引用内容的基础模板 |
| [8.3 代码块.md](8.3%20代码块.md) | 代码块模板（使用 `{% raw %}` 包裹） |

### 第五类：9.x - 模板开发参考

用于模板开发时的字段演示和参考。

| 文件 | 说明 |
|------|------|
| [9. 模板开发-本文档参数.md](9.%20模板开发-本文档参数.md) | 展示当前文档可使用的所有参数和查询方法，包括文档 ID、标题、标签、别名、路径、创建/更新时间等字段的获取方式 |

## 使用说明

1. **子文档框架**：新建文档时选择对应模板，填写摘要和正文内容
2. **父文档目录**：在父文档中插入模板，自动拉取子文档列表和摘要
3. **标签聚合**：修改模板中的 `#指定标签` 为实际标签名称，即可聚合相关内容
4. **代码片段**：直接复制使用或根据需求修改

## 技术说明

模板使用思源笔记的 Action 语法（`.action{}`）和 SQL 查询语句实现动态内容获取，支持：

- 文档属性和内容块的查询
- 循环遍历和条件判断
- 字符串处理和格式化
- 增量更新（避免重复添加）
